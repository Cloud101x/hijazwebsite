# PRD · Atelier Noir (Commerce)

## Problem
A small Lagos atelier releases 18 hand-made pieces per season. The brand reads as editorial and quiet — but every existing Shopify template forces it into a "shop now" voice that destroys the positioning. We need a site that feels more like a *magazine* than a *store*, while still doing the actual transactional work.

## Target audience
1. **Lagos/Abuja design buyers (30–50)** decorating a primary residence or pied-à-terre
2. **Diaspora collectors (35–55)** discovering African-made objects through editorial press
3. **Interior designers** sourcing for client projects, often buying in sets

## Outcomes
- Read like an editorial — first scroll should feel like a magazine cover, not a product grid
- Convert qualified shoppers to bag in ≤ 5 taps from any product card
- Highlight provenance (the makers, the cities) on every piece
- Communicate scarcity quietly — `Last pieces`, `Made to order` badges, no urgency timers or popups

## Non-goals
- Not a marketplace — every piece is curated by the atelier
- No discount codes, flash sales, or "members get 10% off" lock-ins
- Not localised in MVP — NGN-only pricing, English copy only

## Success metrics
| Metric | Target |
| --- | --- |
| Sessions → bag (catalogue view) | ≥ 18% |
| Bag → checkout intent | ≥ 35% |
| Average order value | ≥ ₦220k |
| Newsletter signup on home page | ≥ 4% of sessions |
