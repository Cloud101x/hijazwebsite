# PRD · Sunny & Soul (Bakery + Arcade)

## Problem
A Lagos hospitality concept — bakery in the morning, cocktail bar at night, retro arcade upstairs — needs a single web presence that does three jobs at once: surface the menu, drive reservations, and merchandise the event calendar. Existing restaurant-template websites look generic and undersell the brand.

## Target audience
1. **Yaba locals (25–40)** booking weekend brunch or weeknight after-work drinks
2. **Lagos creatives & tech crowd** who treat the arcade mezzanine as their third place
3. **Tourists and Lagos visitors** discovering the spot via Instagram or a press feature

## Outcomes
- Convert the landing page to a reservation in **≤ 4 taps** on mobile (CTA → drawer → date+time → confirm)
- Surface this week's events on the home page — no separate "events" route
- Make the menu visually inviting enough to be reposted to Instagram
- Communicate the dual identity (bakery **and** arcade) in the first scroll, without splitting them into separate pages

## Non-goals
- Not an ordering / delivery app — walk-in and table reservations only at MVP
- Not a CMS-managed site — menu and events live in `data/*.ts` and ship via PR
- No login / accounts

## Success metrics
| Metric | Target |
| --- | --- |
| Reservations submitted / week | 80+ |
| Tap-to-reservation funnel completion | ≥ 35% |
| Time spent on home page (median) | ≥ 60s |
| Bounce rate (mobile) | < 45% |
